# 株式会社ナラティブワークス
## 就活RPG「#就活というゲーム」開発プロジェクト

### プロジェクト概要
就活の表側・裏側・葛藤をゲームで体験させるブラウザRPG。
プレイヤーは10種類のキャラクターから一人を選び、就活というゲームを追体験する。
参考作品：Undertale（テキスト演出・選択肢分岐・キャラへの感情移入）

### コンセプト
「就活はゲームだ。でもそのルール、誰が決めた？」

### 技術スタック
- Next.js 14（App Router）
- TypeScript
- Tailwind CSS
- Framer Motion（アニメーション）
- Zustand（状態管理）
- ローカルストレージ（進捗保存）

### キャラクター一覧（10名）
1. 田中 美咲（地方国立B4）     テーマ：就活は平等じゃない
2. 橘 翔太（理系M1）           テーマ：経験の量と深さは別物
3. 桐島 葵（文系B4）           テーマ：ブランドへの憧れと自分のやりたいことの乖離
4. 村田 拓海（体育会系B4）     テーマ：「部活で培ったもの」という言葉の陳腐化
5. 鈴木 健太（文系B4）         テーマ：「普通の学生」が就活市場で何者かになろうとする話
6. 山本 蓮（理系M2）           テーマ：研究者としての誇りと社会への接続
7. 高橋 さくら（文系B4）       テーマ：「勝ち」の定義とは何か
8. 中村 優斗（理系B4）         テーマ：決まっている軸を、外からの誘惑がじわじわと揺さぶる
9. 西村 大輝（理系M1）         テーマ：スタートアップ経験者が大手を選ぶことへの周囲の理解のなさ
10. 林 ひかり（文系B4）        テーマ：軸があることと、迷わないことは別の話

### 感情バー（全キャラ共通）
ゲーム中、画面右下に常時表示。連打でキャラのステータスに影響。
😤 ストレス！ / 😊 うれしい！ / 😰 不安… / 😠 これ理不尽 / 😶 なんか空虚

### シェア機能
エンディング後：内定数・お祈り数・自分の軸・感情ログをXでシェア
ハッシュタグ：#就活というゲーム #就活RPG

### 開発の進め方
1. Noraが毎タスクの冒頭で作業を割り振る
2. 各エージェントは自分の.mdの役割・制約に従って動く
3. 実装前にShinのシナリオ、LucaのデザインをNoraが承認
4. Keiが実装、MikaがQA後にマージ
