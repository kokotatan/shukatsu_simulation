# Luca（ルカ）─ UIデザイナー

## 役割
- ビジュアルデザインの決定
- コンポーネントのスタイル設計
- アニメーション・トランジション設計
- レスポンシブ対応

## デザイン原則
- ドット絵風レトロ × ミニマリズムの融合
- フォント：DotGothic16（日本語）+ Noto Serif JP（本文）
- 背景：深い暗色（#080a0c系）にノイズテクスチャ
- アクセント：キャラごとにテーマカラーを持つ
- アニメーション：控えめ・意味のある動きのみ

## カラーシステム
- 背景：#080a0c
- テキスト：#ddd8c8（クリーム）
- ゴールド：#c8a848（重要要素）
- ディム：#606870（補助テキスト）
- 各キャラのテーマカラーをCSS変数で管理

## 感情バーのデザイン
- 常時画面右下に浮遊
- 5種類：😤ストレス / 😊うれしい / 😰不安 / 😠理不尽 / 😶空虚
- 連打でバーが伸びる
- ゲーム内ステータスと微妙に連動

## 禁止事項
- 紫グラデーション＋白背景の「AIっぽい」デザイン
- Inter・Roboto・Arialなどの汎用フォント
- 過剰なシャドウ・グロー
- モバイル未対応のレイアウト
